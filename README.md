# API base project

Install the project:

```sh
$ npm install
$ npm start
```

Default port http://localhost:8080 or .env PORT 

### Docker

```sh
docker build -t {username}/BaseProject:${Version} .
```

```sh
docker run -d -p 80:8080 --restart="always" <youruser>/BaseProject:${Version}
```
